# Веб технології — лабораторні роботи

Цей репозиторій створено для виконання лабораторних робіт з предмету **«Веб технології»**.

## 📌 Мета репозиторію

У цьому репозиторії я буду:

* виконувати лабораторні роботи;
* зберігати вихідний код;
* практикувати навички розробки веб-застосунків;
* вивчати сучасні інструменти та підходи у веб-розробці.

## 🛠️ Технології

Під час виконання лабораторних робіт планується використання таких технологій:

* HTML, CSS
* JavaScript
* Git та GitHub
* (за потреби) фреймворки та бібліотеки

## 📂 Структура

Кожна лабораторна робота буде зберігатися в окремій папці з відповідною назвою.

## 🚀 Прогрес

Репозиторій буде поступово оновлюватися в процесі навчання та виконання завдань.
